## Description
This project analyzes students' performance in three subjects: **Reading, Writing, and Math**. 
The analysis includes:
- Pass/Fail status per subject
- Students passing 2 or more subjects
- Distribution of scores
- Visualization using **Seaborn** and **Matplotlib**

## Dataset
The dataset contains:
- Student demographic info (`gender`, `race/ethnicity`, `parental_level_of_education`)
- Scores (`reading_score`, `writing_score`, `math_score`)
- Derived columns (`pass_reading`, `pass_writing`, `pass_math`, `passed_2_or_more`)
